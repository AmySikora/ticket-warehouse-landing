About the Project

Ticket Warehouse is a synchronization hub that prevents duplicate ticket sales across marketplaces, built with FastAPI, async webhooks, and developer-first design.

The Problem

Sellers often list the same ticket on multiple platforms. When more than one buyer checks out, someone loses — causing refunds, support churn, and reputational damage.

The Solution

A single source of truth for ticket status. Register inventory once, sync everywhere in real-time, and guarantee idempotent sales events.

Stack & Architecture

  • FastAPI backend (Python)
  • Async webhooks with httpx
  • Tests with pytest and respx
  • Deployable via Docker

Roadmap

  • OAuth 2.0 and role-based access
  • PostgreSQL / RDS migration
  • Analytics dashboard
  • Broker portal (React/Angular)
